Week 5

This week I have created a metadata schema and I have implemented this by modifying my MacArthurPark.mei file found in the 'data' folder of my repository. Please refer to this document to see this weeks work in full context. I have enriched my data by using the '@auth.uri' attribute where possible to add firther links to my existing metadata.

My metadata schema contains the following elements:

  • Artist
  • Writer
  • Arranger
  • Encoder
  • Year of Release
  • Publisher
  • Date of Encoding
  • Distributor
